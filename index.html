RISE OF EMPIRES

A Real-Time Strategy Game

Left-click: Select unit/building  |  Drag: Box select
Right-click: Move / Attack  |  WASD or edge: Pan camera

Select Villager → Build Farm (generates points)
Select Villager → Build Barracks → Train Swordsman
Swordsmen auto-attack nearby enemies
AI sends waves every 30s — defend and destroy their base!
